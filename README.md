# Video Overlay and Capture for DOS

This is a video camera live overlay and image capture application for DOS for the JVC TKF-7300U Camera (1994).

## Project Summary

This DOS-based application enables real-time video capture and overlay capabilities using the JVC TKF-7300U Camera from 1994. It's designed for vintage computer enthusiasts and museum installations requiring period-correct video capture solutions. The software supports 640x480 resolution with 24-bit color and saves uncompressed BMP files, making it ideal for archival purposes and historical preservation projects.

## Installation

1. Ensure you have a DOS-compatible system.
2. Download the application files from the repository.
3. Copy the files to your DOS system using a floppy disk or other compatible media.
4. Run the installation script by typing `install.bat` in the DOS command prompt.

## Usage

1. Connect the JVC TKF-7300U Camera to your DOS system.
2. Start the application by typing `capture.exe` in the DOS command prompt.
3. Follow the on-screen instructions to begin live overlay or image capture.

## Features

- Live video overlay from the JVC TKF-7300U Camera.
- Capture still images and save them to disk.
- Simple and intuitive DOS interface.

## System Requirements

- IBM PC compatible computer (386 or better recommended)
- MS-DOS 5.0 or higher
- 640KB RAM minimum
- VGA graphics card
- Available ISA slot for video capture card
- Hard disk with at least 2MB free space
- JVC TKF-7300U Camera with compatible interface cable

## Technical Details

- Video Resolution: 640x480 pixels
- Color Depth: 24-bit
- File Format: BMP (uncompressed)
- Interface: Custom ISA card
- Memory Usage: ~500KB conventional memory
- Capture Speed: Up to 15 frames per second

## Troubleshooting

### Common Issues

1. **No Video Signal**
   - Check camera power and connections
   - Verify ISA card is properly seated
   - Ensure correct I/O address settings

2. **System Crashes**
   - Check CONFIG.SYS for memory conflicts
   - Verify EMM386.EXE settings
   - Try disabling other resident programs

3. **Poor Image Quality**
   - Adjust camera settings
   - Check video cable connections
   - Verify lighting conditions
